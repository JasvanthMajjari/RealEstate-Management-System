🏠 RealEstate Platform – Full Stack MERN Application

A modern and feature-rich full-stack Real Estate web application developed using the MERN Stack. This platform enables users to explore, search, wishlist, and communicate about properties in real-time through a responsive and interactive user interface.

The project is designed with scalable architecture and integrates multiple modern technologies including secure authentication, cloud-based image management, real-time communication, online APIs, and advanced property management features.
